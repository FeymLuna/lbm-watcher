Entry

Weight and BF are smoothed via EWMA (exponentially weighted, half-life ~2.4 days) to dampen bioimpedance noise. The safety guard (target / min intake) computes on the smoothed values; raw values remain visible in the table and chart. BF is optional — if missing, the last known raw value is carried forward.
